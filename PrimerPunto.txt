Cuál es su mayor utilidad

Es una herramienta que nos sirve para la gestionar, administrar, construir  un proyecto en java, ademas de facilitar el envio de archivos jar entre varios proyectos.

Fases de maven

- Validate
- Compile
- Test
- Package
- Integration Test
- Verify
- Install
- Deploy

Ciclo de vida de la construcción

Se encarga de compilar los ficheros fuentes . java

Para qué sirven los plugins

Es una tarea especifica que contribuye a la contruccion de un proyecto.

Qué es y para qué sirve el repositorio central de maven

Es un repositorio que contiene cientos de librerías constantemente actualizadas por sus creadores, esto nos sirve en caso que el repositorio local no cuente con una libreria especifica que necesita, este repositorio central buscara la libreria que se necesita y una vez hecho esto la pondra en el archivo local para que pueda hacer uso de ella.
